# Project Nostalgia

This is a 2D Visual Novel (Unity engine) made as an "ininterpreter to read chapter.txt files" through "6 visual novel layers:
  (1.bakcground, 2.Characters, 3.dialogue, 4.cinematic, 5.foreground. 6.controls)"
  and apply them to the screen with the proper effects & commands, more features can be easy to extends!
  
The system will have a Standalone UI based dialogue system, UI characters & control, audio manager, file handling & encrypting system,
dialogue interpretation system for chapter files(.txt), input, choice screen, text logs, gallery, save&load system.

*(current progress is to make an diary UI & minigame event based on Affection system).

*This project is still IN DEVELOPMENT and also works with Live 2D character as well (if i have time to make characters model).
